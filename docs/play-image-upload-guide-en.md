# Play Image Upload Guide (Generated Assets)

Use these generated files in Play Console listing:

## App icon
- `store-assets/play/icon/play-icon-512.png`
- Size: 512x512

## Feature graphic
- `store-assets/play/feature-graphic/feature-graphic-1024x500.png`
- Size: 1024x500

## Phone screenshots
- Folder: `store-assets/play/screenshots-phone/`
- Files: 8 PNG (`01-...` to `08-...`)
- Size: 1080x1920

## 7-inch tablet screenshots
- Folder: `store-assets/play/screenshots-tablet-7/`
- Files: 6 PNG (`01-...` to `06-...`)
- Size: 1200x1920

## 10-inch tablet screenshots
- Folder: `store-assets/play/screenshots-tablet-10/`
- Files: 6 PNG (`01-...` to `06-...`)
- Size: 1920x1200

## Single zip package
- `store-assets/play/eventi-gare-play-assets.zip`

## Regenerate anytime
```powershell
python scripts/generate-play-assets.py
```
